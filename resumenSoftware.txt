Google Chrome: Un navegador web desarrollado por Google. Es ampliamente utilizado para navegar por Internet y ofrece características como pestañas, extensiones y sincronización entre dispositivos.

Microsoft Word: Un procesador de texto desarrollado por Microsoft como parte de la suite de software Microsoft Office. Es utilizado para crear y editar documentos de texto con funcionalidades avanzadas de formato y diseño.

Adobe Photoshop: Un software de edición de imágenes ampliamente utilizado en diseño gráfico y edición de fotografías. Ofrece una amplia gama de herramientas para retocar, manipular y crear imágenes de alta calidad.
